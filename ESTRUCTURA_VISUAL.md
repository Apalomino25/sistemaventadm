## рҹ”„ CICLO DE VIDA DE DATOS

```
DГӯa 1 - VENTA
в””в”Җ Usuario registra venta
   в””в”Җ INSERT: ventas, detalleventa
   в””в”Җ UPDATE: productos (stock -1)

DГӯa 2 a N - VENTAS ADICIONALES
в””в”Җ MГЎs ventas del mismo patrГіn

DГӯa N+1 - CIERRE
в””в”Җ INSERT: cierres
в””в”Җ Resumen de ventas pagadas
в””в”Җ Resumen de ventas pendientes

Si venta se anula:
в””в”Җ UPDATE: ventas (estado = 0)
в””в”Җ UPDATE: productos (stock +1)
в””в”Җ Historial se mantiene
```
